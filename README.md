# Autonomous Driving AI

## Project Overview
This project focuses on vehicle detection and localization for autonomous driving systems using deep learning and computer vision techniques.

## Dataset
- 3000 labeled vehicle images
- 11 vehicle categories
- Bounding box annotations

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib

## Models Implemented
- Custom CNN
- VGG16
- ResNet50
- MobileNet
- EfficientNetB0

## Results

| Model | Classification Accuracy |
|---------|---------|
| Custom CNN | 52.17% |
| VGG16 | 42.00% |
| ResNet50 | 64.00% |
| MobileNet | 67.17% |
| EfficientNetB0 | 67.00% |

### Best Performing Model
MobileNet achieved the highest classification accuracy of approximately 67.17%.
